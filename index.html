📘 J4DocSearch

J4DocSearch is a high-speed desktop document indexing and search tool for Windows. It monitors your local drives in real time and allows fast keyword-based search across various document formats.

📦 Screen Shot

🚀 How It Works

  • Full Indexing: Scans all local drives to create a search index.
  • Real-Time Monitoring: Automatically updates index on file changes.

🔍 Supported Formats

  • .docx, .xlsx, .pptx
  • .pdf (text-based)
  • .hwpx (Hancom Office)

🖥️ OS Support

Windows 10/11, 64-bit recommended, no admin required.

🌟 Features

  • Standalone EXE, NSIS installer
  • System tray mode, auto-start, duplicate prevention

📘 J4DocSearch

J4DocSearch는 Windows 환경에서 다양한 문서의 내용을 빠르게 검색할 수 있는 고속 데스크톱 인덱싱 도구입니다. 로컬 디스크를 실시간으로 모니터링하며 키워드 기반 검색을 지원합니다.

⬇️ 최신 버전 다운로드

📦 화면 예시

🚀 동작 방식

  • 초기 전체 인덱싱: 모든 로컬 드라이브를 스캔하여 검색 인덱스를 생성합니다.
  • 실시간 감시: 파일 추가/수정/삭제 시 자동으로 인덱스를 업데이트합니다.

🔍 지원 형식

  • .docx, .xlsx, .pptx
  • .pdf (텍스트 기반)
  • .hwpx (한컴오피스)

🖥️ 지원 운영체제

Windows 10 / 11 (64비트 권장), 관리자 권한 없이 실행 가능

🌟 주요 기능

  • 단일 실행파일(EXE), NSIS 설치 프로그램 제공
  • 시스템 트레이 실행, 자동 시작, 중복 실행 방지
